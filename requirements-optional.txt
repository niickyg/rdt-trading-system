# =============================================================================
# NOTES ON PACKAGES IN requirements.txt
# =============================================================================
# The following packages in requirements.txt are worth noting:
#
# - psycopg2-binary: Only required when using a PostgreSQL backend.
#   SQLite (via aiosqlite) is used by default. Safe to omit in SQLite-only deploys.
#
# - graphene / graphql-core: Only required if the GraphQL API endpoint is used
#   (api/graphql/). Not needed for the REST API or web dashboard alone.
#
# - twilio / plyer: Only required if alert notifications are configured.
#   The system runs without them; alerts will be silently skipped if unconfigured.
#
# - APScheduler: Required only for scheduled background tasks (e.g., auto-scan).
#   Not needed if all scans are triggered manually.
#
# - asyncio-throttle: Small utility; only needed if async rate-limiting is used.
#   Currently used in async API call wrappers.
