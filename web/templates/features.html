Machine Learning & Adaptive Learning

  • Per-strategy GradientBoosting models score every signal with a confidence rating
  • Adaptive parameter learning adjusts strategy thresholds based on recent performance
  • Market regime detection (trending, mean-reverting, volatile) influences strategy weights
  • Model drift monitoring with automatic retraining triggers
  • Feature engineering from price action, volume, volatility, and relative strength

Options Trading Module

  • Full options support: calls, puts, vertical spreads, iron condors
  • IV regime-based strategy selection (low IV → debit spreads, high IV → credit spreads)
  • Delta-based strike picking with configurable target deltas
  • IBKR combo order execution for multi-leg strategies
  • Options chain viewer with Greeks display and risk analysis

Risk Management

  • Per-strategy position limits and capital allocation
  • Portfolio-wide max open positions and daily loss caps
  • ATR-based stop loss and profit target calculation
  • Correlation-aware position sizing
  • VIX filter prevents entries during extreme market stress
  • Automatic trade rejection with logged reasons

Real-Time Dashboard

  • Dark-themed professional interface with WebSocket live updates
  • Live positions with current price, P&L, and stop/target progress bars
  • Signal feed with confidence scores, RRS values, and strategy tags
  • Trade journal with MFE/MAE analysis and performance breakdowns
  • Strategy analytics with per-strategy win rates and P&L
  • Built-in backtester with parameter optimization
  • Agent health monitoring and system status

Broker Integration

  • Interactive Brokers (IBKR) as sole data source and execution broker
  • Real-time quotes and historical bar data with intelligent caching
  • Automatic order placement with bracket orders (stop + target)
  • Paper trading mode for strategy development and testing
  • Position reconciliation on system restart
