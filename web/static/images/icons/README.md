# RDT Trading System - PWA Icons

This directory should contain the following icon files for the Progressive Web App:

## Required Icons

### Standard Icons (PNG format)
- `icon-16x16.png` - Favicon small
- `icon-32x32.png` - Favicon standard
- `icon-72x72.png` - Small icon
- `icon-96x96.png` - Badge icon
- `icon-128x128.png` - Medium icon
- `icon-144x144.png` - Android icon
- `icon-152x152.png` - iOS icon
- `icon-167x167.png` - iPad Pro icon
- `icon-180x180.png` - iPhone icon
- `icon-192x192.png` - PWA required (Android)
- `icon-384x384.png` - Large icon
- `icon-512x512.png` - PWA required (splash)

### Maskable Icons (for Android adaptive icons)
- `icon-maskable-192x192.png` - Maskable version
- `icon-maskable-512x512.png` - Maskable version

### Safari/Apple
- `safari-pinned-tab.svg` - Safari pinned tab icon (monochrome SVG)

### Windows
- `browserconfig.xml` - Windows tile configuration

## Icon Generation

You can generate all required icons from a single high-resolution source image (1024x1024 recommended) using tools like:

1. **PWA Asset Generator**: https://github.com/nicosapp/pwa-asset-generator
2. **RealFaviconGenerator**: https://realfavicongenerator.net/
3. **PWA Builder**: https://www.pwabuilder.com/imageGenerator

## Design Guidelines

- Use a simple, recognizable design
- Ensure the icon is legible at small sizes
- For maskable icons, keep important content within the safe zone (inner 80%)
- Use the RDT Trading brand colors:
  - Primary: #1a1a2e
  - Accent: #4CAF50
  - Background: #f0f2f5

## Suggested Design

A simple "RDT" text or trading chart icon in green (#4CAF50) on a dark blue (#1a1a2e) background works well for this application.
